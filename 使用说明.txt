文件重命名工具 - 映射组件
============================

快速开始：
---------
1. 双击 "启动映射工具.bat" 或 "启动映射工具.ps1"
2. 选择要运行的程序
3. 按照界面提示操作

可执行文件说明：
---------------
- 启动映射工具.bat          - 主启动器（批处理版本）
- 启动映射工具.ps1          - 主启动器（PowerShell版本，功能更丰富）
- run_mapping_tool.bat      - 直接运行最终版应用
- run_demo.bat              - 直接运行演示程序
- run_test.bat              - 直接运行测试程序
- 安装检查.bat              - 检查Python环境和依赖

系统要求：
---------
- Windows 7/8/10/11
- Python 3.7 或更高版本
- 无需额外安装其他软件

如果遇到问题：
-------------
1. 运行 "安装检查.bat" 检查环境
2. 确保Python已正确安装并添加到PATH
3. 确保所有.py文件完整
4. 检查文件权限

功能特点：
---------
✓ 创建key-value映射规则
✓ 批量替换文件名中的特定文本
✓ 支持正则表达式
✓ 配置保存和加载
✓ 直观的图形界面
✓ 实时预览效果

使用示例：
---------
将 "IMG_20231201_143022.jpg" 重命名为 "照片_2023年12月01日 143022.jpg"

映射规则：
- IMG_ → 照片_
- 2023 → 2023年
- 12 → 12月
- 01 → 01日
- _ → (空格)

技术支持：
---------
如有问题，请检查：
1. Python版本是否符合要求
2. 所有文件是否完整
3. 是否有足够的文件权限

祝您使用愉快！
