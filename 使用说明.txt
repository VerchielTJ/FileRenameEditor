文件重命名工具 - 使用说明
============================

快速开始：
---------
1. 首次使用：双击 "环境设置.bat" 或 "环境设置.ps1" 设置运行环境
2. 运行程序：双击 "运行程序.bat" 或 "运行程序.ps1"
3. 或者直接运行：python main.py

项目结构：
---------
FileRenameEditor/
├── app.py                          # 主应用程序入口
├── main.py                         # 程序启动入口
├── setup.py                        # 安装脚本
├── models/                         # 数据模型层
│   └── file_manager.py            # 文件管理器
├── views/                          # 视图层
│   ├── main_window.py             # 主窗口界面
│   └── components/                # UI组件
│       └── mapping_widget.py      # 映射组件
├── controllers/                    # 控制器层
│   └── rename_controller.py       # 重命名控制器
├── 环境设置.bat/.ps1              # 环境设置脚本
├── 运行程序.bat/.ps1              # 程序运行脚本
└── 其他配置和文档文件

系统要求：
---------
- Windows 7/8/10/11
- Python 3.7 或更高版本
- 无需额外安装其他软件

环境设置：
---------
首次使用请运行环境设置脚本：
- 环境设置.bat (批处理版本)
- 环境设置.ps1 (PowerShell版本，功能更丰富)

环境设置脚本会：
✓ 检查Python环境
✓ 验证必要模块
✓ 创建虚拟环境
✓ 安装项目依赖
✓ 激活虚拟环境

功能特点：
---------
✓ 添加前缀和后缀
✓ 创建key-value映射规则
✓ 批量替换文件名中的特定文本
✓ 实时预览效果
✓ 直观的图形界面
✓ 完善的错误处理

使用示例：
---------
将 "IMG_20231201_143022.jpg" 重命名为 "照片_2023年12月01日 143022.jpg"

映射规则：
- IMG_ → 照片_
- 2023 → 2023年
- 12 → 12月
- 01 → 01日
- _ → (空格)

操作流程：
---------
1. 启动程序
2. 选择工作文件夹
3. 设置重命名规则（前缀、后缀、映射）
4. 预览重命名效果
5. 执行重命名操作

运行方式：
---------
方式一：使用运行脚本（推荐）
- 双击 "运行程序.bat" 或 "运行程序.ps1"

方式二：命令行运行
- 激活虚拟环境：venv\Scripts\activate.bat
- 运行程序：python main.py

方式三：直接运行
- python main.py（需要先设置环境）

如果遇到问题：
-------------
1. 运行 "环境设置.bat" 重新设置环境
2. 确保Python已正确安装并添加到PATH
3. 确保所有.py文件完整
4. 检查文件权限
5. 查看错误信息并参考故障排除指南

注意事项：
---------
1. 执行重命名前请备份重要文件
2. 使用预览功能验证重命名规则
3. 确保对目标文件夹有写权限
4. 程序会自动跳过已存在的文件名

技术支持：
---------
如有问题，请检查：
1. Python版本是否符合要求
2. 所有文件是否完整
3. 是否有足够的文件权限
4. 虚拟环境是否正确设置

祝您使用愉快！
