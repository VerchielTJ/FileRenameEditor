文件重命名工具 - 简单使用说明
=====================================

快速开始：
---------
1. 双击 "启动工具.bat" 或 "启动工具.ps1"
2. 或者直接运行：python simple_main.py

系统要求：
---------
- Windows 7/8/10/11
- Python 3.7 或更高版本
- 无需额外安装其他软件

功能特点：
---------
✓ 添加前缀和后缀
✓ 创建key-value映射规则
✓ 批量替换文件名中的特定文本
✓ 实时预览效果
✓ 直观的图形界面
✓ 完善的错误处理

前缀和后缀说明：
--------------
- 前缀：会添加到文件名的开头
- 后缀：会添加到扩展名之前
- 例如：原文件 "document.pdf"
  - 前缀 "重要_" → "重要_document.pdf"
  - 后缀 "_v1" → "document_v1.pdf"
  - 前缀 "重要_" + 后缀 "_v1" → "重要_document_v1.pdf"

使用示例：
---------
将 "IMG_20231201_143022.jpg" 重命名为 "照片_2023年12月01日 143022.jpg"

方法一：使用前缀和后缀
- 前缀：照片_
- 后缀：_2023年12月01日

方法二：使用映射规则
- IMG_ → 照片_
- 2023 → 2023年
- 12 → 12月
- 01 → 01日
- _ → (空格)

操作流程：
---------
1. 启动程序
2. 选择工作文件夹
3. 设置重命名规则（前缀、后缀、映射）
4. 预览重命名效果
5. 执行重命名操作

如果遇到问题：
-------------
1. 确保Python已正确安装并添加到PATH
2. 确保所有.py文件完整
3. 检查文件权限
4. 查看错误信息

注意事项：
---------
1. 执行重命名前请备份重要文件
2. 使用预览功能验证重命名规则
3. 确保对目标文件夹有写权限
4. 程序会自动跳过已存在的文件名

祝您使用愉快！
